HONK

Honk is designed to replace native javascript alert and confirm dialogs. Callbacks are supported on both alerts and confirms. jQuery is required. SaSS files are included and utilize Bourbon.io, but the compiled css file could be used as well. Honk is released under the MIT license.

Alerts

honk.alert('Your message goes here!', function(){
	//Alerts & Confirms support callbacks if needed.
});

Confirms

honk.confirm('Your message goes here!', function(){
	//Alerts & Confirms support callbacks.
});
